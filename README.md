В данной лабораторной работе было изучено восстановление работоспособности программы с использованием системы контроля версий Git. Проект из первой лабораторной работы был скопирован в новый репозиторий. В программу была внесена ошибка путём удаления модуля math_operations.py. Работоспособность нарушилась — программа выдавала ошибку импорта. После этого был выполнен откат к последней рабочей версии через историю коммитов, и модуль был восстановлен. Восстановленная программа проверена и загружена в репозиторий. В результате освоены навыки отката изменений и восстановления работоспособности проекта средствами Git.
